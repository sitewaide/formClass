--Français Version : Alpha 0.3 Créateur : Site-Waide.fr Site : http://site-waide.fr Date de création : 27/06/2012 00:20

FormClass est une classe PHP vous permettant de créer rapidement et simplement vos formulaires html tout en gardant le contrôle sur votre code source. Les fichiers sont commentés en Anglais pour une meilleur diffusion. Fonctionnalités : -Création de la balise form -Création des différents type de balises input, sauf select et radio. (checkbox partiellement pris en compte) -Fermeture de la balise form et boutton submit

Fonctionnalités à venir : -Prise en charge des checkbox complète. -Prise en charge des selects -Prise en charge des textareas -prise en charge des radios

FormClass is PHP class allowing you to quickly and easily create your html forms while you keep the control of your source code. Files are commented in English for a better diffusion Features : -Form tag creation -Creation of the input tag, except select and radio ( checkbox are partially implemented) -form tag close and submit button Forthcoming features : -Complete implementation of checkbox -Complete implementation of selects -Complete implementation of textareas -Complete implementation of radios
